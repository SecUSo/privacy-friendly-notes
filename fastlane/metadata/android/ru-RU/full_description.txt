Свободное приложение для заметок. Оно из группы приложений дружественных к конфиденциальности (Privacy Friendly Apps, PWA) разработанными исследовательской группой SECUSO в Институте Технологии Карлсруэ (KIT). Подробнее читайте на <a href="https://secuso.org/pfa">secuso.org</a>.

Доступны следующие четыре типа заметок:
1. Простые текстовые заметки
2. Контрольный список
3. Аудиозаметки
4. Зарисовки
